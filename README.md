# Inadea | Institución Médica Líder en Cardioprotección

Este es un proyecto de [Next.js](https://nextjs.org) creado para Inadea, enfocado en servicios de salud cardiovascular y áreas cardioasistidas.

## Comenzando

Primero, ejecuta el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

Puedes comenzar a editar la página modificando `app/page.tsx`. La página se actualiza automáticamente a medida que editas el archivo.

Este proyecto utiliza [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) para optimizar y cargar las fuentes tipográficas automáticamente.
